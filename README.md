# My GNOME Visual Setup (CustomPanel)

Скрипт полностью скопирует мой визуальный стиль (обои, тему Shell `CustomPanel`, иконки, курсоры, настройки `.dconf` и всё необходимое) на любую другую машину на GNOME.

Я сделал этот пакет так, чтобы его установка была абсолютно безопасной — вы можете в 1 клик сделать полную копию своих текущих настроек перед тем, как ставить мои!

## 🚀 Установка

Установка состоит всего из скачивания репозитория и запуска распаковки.

Откройте терминал и выполните:

```bash
# 1. Скачиваем репозиторий с темой на ваш ПК
git clone https://github.com/YOUR_USERNAME/gnome-visual-transfer.git
cd gnome-visual-transfer

# 2. ДЕЛАЕМ БЭКАП (Необязательно, но очень рекомендую)
# Сохранит ваши текущие обои и настройки на Рабочий стол
bash backup.sh ~/Desktop

# 3. ПРИМЕНЯЕМ МОЮ ТЕМУ
bash restore.sh my-theme.tar.gz
```

*После восстановления, чтобы расширения и верхняя панель применились окончательно — перезайдите в систему (выйти/зайти).*

---

## 🛠️ Как откатиться назад (Если не понравилось)

Если вы делали Шаг 2 (бэкап) перед установкой, вернуть всё как было можно за пару секунд.

1. Зайдите в папку с репозиторием
2. Запустите скрипт восстановления, но укажите ему ВАШ файл с рабочего стола:

```bash
cd ~/gnome-visual-transfer
bash restore.sh ~/Desktop/gnome-visual-backup-*.tar.gz
```
Ваша старая система вернётся ровно в то состояние, в котором была.

## 📦 Как мне (автору) обновить тему в репозитории?

Если вы поменяли обои или цвета в теме у себя на компьютере и хотите обновить этот репозиторий на GitHub:

1. Создайте свежий архив своих текущих настроек командой:
   ```bash
   bash backup.sh ./
   ```
2. Скрипт создаст файл `gnome-visual-backup-XXXXXX.tar.gz`. Удали старый `my-theme.tar.gz` и переименуй новый:
   ```bash
   rm my-theme.tar.gz
   mv gnome-visual-backup-*.tar.gz my-theme.tar.gz
   ```
3. Сделайте `git push` изменения в GitHub:
   ```bash
   git add my-theme.tar.gz
   git commit -m "Update theme"
   git push
   ```
